# `Cadastro`
_aplicação de cadastro simples, com funções de | criar | editar | apagar | paginação |_

## Requer:

[**MySQL Server**](https://dev.mysql.com/downloads/mysql/) <br>
[**YII2 Framework**](https://www.yiiframework.com) 

<br>

### Configuração:

* Importe o banco e tabelas, com o arquivo `dynamikadb.sql`

* Edite o arquivo config/db.php para conectar ao seu servidor MySQL:

  `username` => login<br>`password` => senha
